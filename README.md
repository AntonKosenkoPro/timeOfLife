# Time of Life

Идея приложения - максимально простой трекинг своего времени и аналитика, куда это время уходит. Простой в том смысле, что требующий минимальных усилий и временных затрат.

Как достигается простота:
- Максимальная интеграция в систему (iOS) - виджеты, шорткаты, экран блокировки
- Оптимизированный под добавление трек-записей UX
- Интеграции с другими приложениями и API для автоматического подтягивания потраченного времени (Garmin, Fitness, Screen Time, Steam etc.)
- Интеграция с другими устройствами, которые могут дать данные для тайм-трекинга (Mac, Windows etc.)
